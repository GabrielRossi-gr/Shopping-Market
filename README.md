### Comandos Para Rodar:

### Comando para rodar o front-end
```
yarn dev                                //Roda em modo desenvolvedor
  ou
npm run dev
```

### Comando para rodar o banco de dados de teste
```
cd src/json-server                      // Entrar na pasta do "banco de dados"         
npx json-server --watch db.json		// Rodar  
```
